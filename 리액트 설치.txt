1. Node.js 설치
Node.js와 npm의 설치 여부를 확인하려면 아래 명령어를 사용한다.

bash
node -v  # Node.js 버전 확인
npm -v   # npm 버전 확인


2. React 프로젝트 생성
npx create-react-app 명령어로 React 프로젝트를 생성할 수 있다.

bash
npx create-react-app my-app


- my-app은 프로젝트 이름이다.


3. 프로젝트 디렉터리로 이동
프로젝트 생성 후, 디렉터리로 이동하여 개발 서버를 실행한다.

bash
cd my-app
npm start


*npm create vite 사용법*
npm create vite는 Vite 프로젝트를 생성하는 명령어이다.

1. Vite 프로젝트 생성
bash
npm create vite@latest


2. 프로젝트 이름과 템플릿 선택
프로젝트 이름과 사용할 템플릿을 선택한다. React 템플릿을 선택하면 React 프로젝트가 설정된다.


3. 프로젝트 디렉터리로 이동
bash
cd my-vite-app


4. 의존성 설치
bash
npm install


5. 개발 서버 실행
bash
npm run dev


npm create vite vs npx create-react-app

- create-react-app: React 전용으로, 설정이 간단하지만 빌드 속도가 느리다.
- vite: 빠른 개발 속도와 효율적인 빌드를 제공하며, React뿐만 아니라 여러 프레임워크도 지원한다.


결론
- create-react-app은 React 전용으로 빠르게 시작할 수 있다.
- vite는 빠른 빌드와 개발 환경을 제공하여, 대규모 프로젝트에서 유리하다.
